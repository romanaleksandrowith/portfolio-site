# Roman Automation Portfolio

Статический одностраничный сайт-презентация проектов для малого бизнеса.

## Что внутри

- `index.html` — структура страницы и четыре смысловых раздела.
- `style.css` — адаптивный темный дизайн с яркими синими и голубыми акцентами.
- `script.js` — переключение разделов, RU/EN язык и сохранение языка в `localStorage`.
- `assets/presentations/` — PDF-презентации проектов.

## Разделы

- Обзор
- Оптимизатор маршрутов
- AI-отчеты и PDF
- Обо мне и контакты

## Локальный запуск

Сайт не требует сборки и дополнительных зависимостей.

```bash
python -m http.server 8000
```

После запуска откройте:

```text
http://localhost:8000
```

Проект готов для публикации через GitHub Pages.
